# Wonderful-Color

[English](./README.md) | [中文](./README_zh.md)

## 项目简介

Wonderful-Color 是一个轻量级的网页工具，用于生成和探索颜色组合。项目参考了 Coolors 等配色工具，重点在于通过实际开发来学习现代前端技术，并实现具有一定实用价值的功能。

该项目强调简洁的设计、即时反馈的交互，以及基于规则的配色生成逻辑。

## 功能特性

- 支持点击按钮或按空格键生成新的配色方案
- 支持锁定指定颜色，在刷新时保持不变
- 智能配色生成逻辑：
  - 无锁定时：生成渐变色或互补色组合
  - 有锁定时：基于锁定颜色生成色相相近的配色
- 基于亮度算法自动调整文字颜色，保证可读性
- 全屏响应式布局，使用竖向色条展示
- 实时显示颜色的十六进制值

## 技术栈

- Vue 3（Composition API）
- chroma.js（颜色计算与插值）

## 实现原理

### 配色生成逻辑

- 当没有锁定颜色时：
  - 随机选择以下两种策略之一：
    - 生成两种随机颜色的渐变色（scale）
    - 生成一对互补色（complementary）
  - 使用 `chroma.scale(...).mode('lch')` 保证颜色过渡更加自然

- 当存在锁定颜色时：
  - 以第一个锁定颜色为基准
  - 在其色相（Hue）附近进行扰动（±20°）
  - 对饱和度（Saturation）和亮度（Lightness）进行小幅随机调整
  - 生成风格统一的配色方案

### 文字对比度处理

为了保证文字在不同背景下的可读性，使用 YIQ 亮度算法：
```

Y = (R × 299 + G × 587 + B × 114) / 1000

```
- 当 Y ≥ 128 时，使用黑色文字
- 当 Y < 128 时，使用白色文字


## 使用方法

1. 在浏览器中打开页面
2. 按下空格键或点击 “Refresh Colors” 按钮生成新配色
3. 点击锁图标可以锁定当前颜色
4. 再次生成配色时，已锁定颜色不会改变
